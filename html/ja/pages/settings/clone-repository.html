✂｜⑥レポジトリのクローン

1. githubプロフィールページの『"Repositories"->"New"』から空のプライベートレポジトリを作成してください.

自由に記載してください
Public/Private: Private
Check "Add a README.md file" checkbox
-> Create repository

警告

privateバッチがついていることを必ず確認してください. ついてない場合、作成したレポジトリから『"Setting"->"Change visibility"』でprivateに変更してください.

2. アカウントのメニューから『"Settings"->"Developer Settings"->"Personal access tokens"->"Generate new token"』

3. パーソナルアクセストークンを生成し、その文字列をコピーしてください

Note: PAT
Check "workflow" checkbox
-> Generate token

作成したレポジトリから"Settings"->"Secrets"->"Actions"->"New repository secret"』

Name: PAT
Paste the string of personal access token.
-> Add secret

5. 作成したレポジトリにアクセスし、『"Actions"->"set up workflow yourself"->"Start commit"』でmain.yamlを作成してください.

6. .github/workflowにmain.yamlがあることを確認し、workflow_clone.yamlをアップロードしてください.

7. ファイルをアップロード後"Commit changes"をクリックしてください.

8. 作成したレポジトリから『"Actions"->"clone apisify"->"Run workflow"->"Select a branch"->"Run workflow"』
