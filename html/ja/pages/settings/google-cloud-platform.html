🥯｜③Google Cloud Platformの設定

1. 新規プロジェクトを作成してください.

注釈

Since apisify uses Google Cloud Platform, it is recommended that you create a new account.

2. Google Cloud Platformを開き、左上の▼をクリックしてください.

https://console.cloud.google.com/

3. 新規プロジェクトを作成してください.

4. "BigQuery API"を検索してください.

5. "BigQuery API"を有効化してください.

6. サービスアカウントを作成してください.

https://console.cloud.google.com/iam-admin/serviceaccounts/create

注釈

ロールは"BigQuery管理者"を付与してください.

7. 作成したサービスアカウントのキー(json)を作成してください.
