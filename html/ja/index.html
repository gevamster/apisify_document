apisifyドキュメントへようこそ！

😺｜①GitHubアカウント作成
GitHubアカウントの作成方法を説明します.
GitHub Actionsを使用するため、アカウントを新規作成することを推奨します.
所要時間：3~6min
👌｜②Oktetoアカウント作成
Oktetoアカウントの作成方法及びトークンの取得方法を説明します.
所要時間：1~3min
🥯｜③Google Cloud Platformの設定
Google Cloud Platformアカウントの作成方法及びBigQueryの有効化を説明します.
所要時間：5~10min
🐂｜④apisifyアカウント作成
apisifyアカウントの作成方法を説明します.
所要時間：1~3min
🗝｜⑤apisifyライセンスキーの作成
apisifyのライセンスキー取得方法を説明します.
所要時間：1min
✂｜⑥レポジトリのクローン
apisifyのレポジトリをクローンします.
所要時間：5~10min
🔌｜⑦連携
apisifyの稼働方法を説明します.
所要時間：3~6min
🤖｜⑧新規注文
apisifyでの注文方法を説明します.
所要時間：1min

設定:

  • 😺｜①GitHubアカウント作成
    • 1. GitHubを開き、"Sign up"をクリックしてください.
    • 2. 必要な項目を入力してください.
    • 3. 設定したメールアドレスに届くコードを入力してください.
    • 4. メンバー数はJust meを設定してください.
    • 5. 興味のある項目を自由に選択してください.
    • 6. Freeプランを選択してください.
    • 7. GitHubアカウントの作成は完了しました.
  • 👌｜②Oktetoアカウント作成
    • 1. Oktetoを開き、"Try Free"をクリックしてください.
    • 2. github で作成したGitHubアカウントでログインしてください.
    • 3. Oktetoアカウントの作成は完了しました.
  • 🥯｜③Google Cloud Platformの設定
    • 1. 新規プロジェクトを作成してください.
    • 2. Google Cloud Platformを開き、左上の▼をクリックしてください.
    • 3. 新規プロジェクトを作成してください.
    • 4. "BigQuery API"を検索してください.
    • 5. "BigQuery API"を有効化してください.
    • 6. サービスアカウントを作成してください.
    • 7. 作成したサービスアカウントのキー(json)を作成してください.
  • 🐂｜④apisifyアカウント作成
    • 1. apisifyを開き、プランを選択してください.
    • 2. 必要な項目を入力してください.
  • 🗝｜⑤apisifyライセンスキーの作成
    • 1. "apisify manager"にDMでstripeに登録したメールアドレスを以下のコマンドのように送付してください.
  • ✂｜⑥レポジトリのクローン
    • 1. apisifyのgithubページにログインし、『"Code" -> "Download ZIP"』からファイルをダウンロードしてください.その後、このファイルを解凍してください.
    • 2. githubプロフィールページの『"Repositories"->"New"』から空のプライベートレポジトリを作成してください.
    • 3. 作成したレポジトリから『"Add file"->"Upload files"』とアクセスし、先程解凍したフォルダ内のファイルをドラッグアンドドロップによりアップロードしてください.
    • 4. 作成したレポジトリにアクセスし、『"Actions"->"set up workflow yourself"->"Start commit"』でmain.yamlを作成してください.
    • 5. .github/workflowにmain.yamlがあることを確認し、『"Add file"->"Upload files"』から解凍したフォルダ内の.github/workflows直下にあるworkflow_auth.yaml及びworkflow_deploy.yamlをアップロードしてください.
    • 6. "Add a README file"にチェックをつける.
    • 7. ファイルをアップロード後"Commit changes"をクリックする.
  • 🔌｜⑦連携
    • 1. Oktetoにログインし、『"Settings" -> "Developer Settings" -> "New Tokens". / "Settings" -> "Developer Settings" -> "New Tokens"』からトークンを作成してください.
    • 2. GitHubにログインし、『"Settings" -> "Secrets" -> "Actions" -> "New repository secret"』と進み、以下の通り登録してください.
    • 3. Oktetoにログインし、"Launch Dev Environment"をクリックする.
    • 4. "Only select repositories"からapisifyレポジトリを選択し、デプロイする.
    • 5. apisifyviewに記載のURL(/viewがついてない方)をクリックしてください.
    • 6. ログインしてください.初期設定は以下のとおりです.
    • 7. ユーザー名及びパスワードを変更してください.
    • 8. apisify-licensekey で作成したapisifyライセンスキーを登録してください.
    • 9. 取引所のapiキーとシークレットキーを登録してください.
