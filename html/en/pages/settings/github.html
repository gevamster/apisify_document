Create a GitHub account

Note

Since apisify uses GitHub Actions, it is recommended that you create a new account.

1. Access GitHub and click “Sign up”.

https://github.com/

2. Enter the required information.

3. Enter the code that will be sent to the email address you set.

4. Set the number of members to “Just me”.

5. Feel free to select items of interest.

6. Select Free plan.

7. You have now created a GitHub account.
