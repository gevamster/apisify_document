✂｜⑥Clone repository

Note

Omit this step if you are using the free plan.

1. Access to github page of apisify and Download files by 『”Code” -> “Download.ZIP”』. And, unzip this file.

https://github.com/gevamster/apisify

2. Create an empty private repository from 『”Repositories”->”New”』 on your github profile page.

Repository name: describe freely
Public/Private: Private
-> Create repository

Note

Be sure to check that the private batch is attached. If not, change the batch to private by clicking 『”Setting”->”Change visibility”』 in the repository you created.

3. From the repository you have created, go to 『”Add file”->”Upload files”』 and upload the files in the folder you have just unzipped by drag and drop.

4. Access the repository you created and create main.yaml by selecting “Actions”->”set up workflow yourself”->”Start commit”.

5. Make sure you have main.yaml in .github/workflow, and then go to 『”Add file”->”Upload files”』 and upload “workflow_auth.yaml” and “workflow_deploy.yaml” in .github/workflows in the folder you have just unzipped.

6. Check the “Add a README file” checkbox.

7. Click “Commit changes” after uploading files.
