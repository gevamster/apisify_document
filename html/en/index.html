Settings:

  • 😺｜①Create a GitHub account
    • 1. Access GitHub and click “Sign up”.
    • 2. Enter the required information.
    • 3. Enter the code that will be sent to the email address you set.
    • 4. Set the number of members to “Just me”.
    • 5. Feel free to select items of interest.
    • 6. Select Free plan.
    • 7. You have now created a GitHub account.
  • 👌｜②Create a Okteto account
    • 1. Access Okteto and click “Try Free”.
    • 2. Login with the GitHub account you created in github.
    • 3. You have now created a Okteto account.
  • 🥯｜③Settings of Google Cloud Platform
    • 1. Create a Google account
    • 2. Access Google Cloud Platform and click ▼.
    • 3. Create new project.
    • 4. Search “BigQuery API”.
    • 5. Activate “BigQuery API”.
    • 6. Create Service account.
    • 7. Create a key(json) for the service account you created.
  • 🐂｜④Create an apisify account
    • 1. Access apisify and select plan.
    • 2. Enter the required information.
  • 🗝｜⑤Create license key of apisify
    • 1. Send a DM to “apisify manager” with the email address resistered to stripe as shown in the following command.
  • ✂｜⑥Clone repository
    • 1. Access to github page of apisify and Download files by 『”Code” -> “Download.ZIP”』. And, unzip this file.
    • 2. Create an empty private repository from 『”Repositories”->”New”』 on your github profile page.
    • 3. From the repository you have created, go to 『”Add file”->”Upload files”』 and upload the files in the folder you have just unzipped by drag and drop.
    • 4. Access the repository you created and create main.yaml by selecting “Actions”->”set up workflow yourself”->”Start commit”.
    • 5. Make sure you have main.yaml in .github/workflow, and then go to 『”Add file”->”Upload files”』 and upload “workflow_auth.yaml” and “workflow_deploy.yaml” in .github/workflows in the folder you have just unzipped.
    • 6. Check the “Add a README file” checkbox.
    • 7. Click “Commit changes” after uploading files.
  • 🔌｜⑦Integration
    • 1. Login to Okteto and Create a personal access token by 『”Settings” -> “Developer Settings” -> “New Tokens”. / Create a token by “Settings” -> “Developer Settings” -> “New Tokens”』.
    • 2. Login to GitHub, go to “Settings” -> “Secrets” -> “Actions” -> “New repository secret” and register as follows.
    • 3. Login to Okteto and Click “Launch Dev Environment”.
    • 4. Click your apisify repository from “Only select repositories” and Deploy.
    • 5. Click on the URL listed in the apisifyview.
    • 6. Please login. The default settings are as follows.
    • 7. Please change your username and password.
    • 8. Register the apisify license key created in apisify-licensekey.
    • 9. Register your api key and secret key for the exchange.

Sample:

  • 🚀｜⑨Recurring order by apisify flow
    • 1. Login to Okteto and Click on the URL listed in the apisifyflow.
    • 2. Click “New blank workflow”.
    • 3. Click “Import from file” and select “recurring_buy.json”.
    • 4. Click node Key and input apikey and secretkey.
    • 5. Click node URL and input apisify trade URL “<YOUR SERVER URL>/v1/order”.
    • 6. Click node discord and input discord URL.
    • 7. Input the other parameters.
    • 8. Set Cron and turn Active.
    • 9. Click Execute Workflow for test
